🎯 Reto: Saltar Obstáculos

📝 Descripción del reto:

Crea un personaje que avance continuamente y salte con la barra espaciadora para evitar obstáculos.

  • ✅ Usa “cuando se presione la bandera verde”
  • ✅ Usa “mover 10 pasos” continuamente
  • ✅ Usa “cuando se presione tecla espacio” para saltar
  • ✅ Usa “cambiar y” o “fijar y” para hacer el salto
  • ✅ Usa “tocar obstáculo” para detectar colisión

📚 ¿Cómo se evalúa este reto?

El reto se evalúa con una rúbrica de 5 criterios. Por cada criterio cumplido, obtendrás 20 puntos.

  • ✅ 5 criterios → 100 puntos → ¡Reto superado!
  • 🟡 3-4 criterios → 60-80 puntos → A mejorar
  • 🔴 Menos de 3 criterios → Sigue practicando
